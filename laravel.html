Pengertian Laravel

Laravel adalah framework PHP open-source yang digunakan untuk membangun aplikasi web dengan sintaks yang elegan, sederhana, dan ekspresif. Laravel menyediakan berbagai fitur seperti routing, middleware, ORM, autentikasi, dan masih banyak lagi.

Bahan Pendukung

Untuk menginstal Laravel, Anda membutuhkan:

  • PHP >= 8.0
  • Composer
  • Web Server (Apache/Nginx)
  • Database (MySQL/PostgreSQL)
  • Node.js & NPM (untuk frontend jika diperlukan)

Instalasi Laravel

Ikuti langkah berikut untuk menginstal Laravel:

  1. Pastikan PHP, Composer, dan Node.js telah terinstal.
  2. Jalankan perintah berikut untuk membuat proyek Laravel:
    3. composer create-project laravel/laravel myproject
  3. Masuk ke direktori proyek dan jalankan server:
    4. cd myproject
php artisan serve
  4. Set konfigurasi database di file .env.
  5. Jalankan migrasi database:
    6. php artisan migrate
  6. Jika menggunakan Vue atau React, instal dependensi frontend:
    7. npm install && npm run dev

Routing di Laravel

Routing di Laravel ditentukan dalam file routes/web.php. Contoh route sederhana:

Route::get('/', function () {
    return view('welcome');
});

Contoh route dengan controller:

use App\Http\Controllers\UserController;

Route::get('/users', [UserController::class, 'index']);

Struktur Proyek Laravel

Laravel memiliki struktur proyek sebagai berikut:

  • app/ - Berisi logika aplikasi
  • routes/ - Berisi konfigurasi routing
  • database/ - Berisi migrasi dan seeder
  • resources/ - Berisi tampilan dan asset frontend
  • public/ - Direktori utama yang dapat diakses oleh pengguna

Middleware di Laravel

Middleware digunakan untuk memfilter permintaan HTTP. Contoh membuat middleware baru:

php artisan make:middleware CheckUser

Middleware dapat didaftarkan dalam app/Http/Kernel.php.

Deploy Laravel

Untuk melakukan deploy Laravel ke server, ikuti langkah berikut:

  1. Upload kode ke server menggunakan Git atau FTP.
  2. Pastikan semua dependency terinstal dengan perintah:
    3. composer install --optimize-autoloader --no-dev
  3. Set permission pada folder storage dan bootstrap/cache:
    4. chmod -R 775 storage bootstrap/cache
  4. Konfigurasi file .env dan jalankan migrasi database:
    5. php artisan migrate --force
  5. Pastikan konfigurasi queue dan cron job telah diatur jika diperlukan.
  6. Gunakan supervisor untuk menjalankan queue worker di latar belakang.
